# Fiscal Code's request

The Italian Tax Code (“ **Codice Fiscale**”) is a personal alphanumeric code (combination of numbers and letters) used to recognize individuals in Italy. It is required when dealing with public offices and/or the public administration.

The “Codice Fiscale” is not compulsory. However, incoming exchange students need to obtain one to open a bank account, sign specific contracts (i.e. phone line), register for the National Healthcare Service (SSN). **Please note**: according to the Italian Revenue Agency guidelines, exchange students who are not registered as temporary residents in Milano are [**not** required](https://www.yesmilano.it/en/study/how-to/get-italian-tax-code-codice-fiscale#paragraph-11861) to obtain a fiscal code in order to sign a rental contract.

The Codice Fiscale must be requested **after** arrival in Italy. Exchange students can follow the application procedure at this [link](https://www.yesmilano.it/en/study/how-to/get-italian-tax-code-codice-fiscale).

The fiscal code application procedure is **free**.

Please note that Bocconi University does not provide exchange students with a  codice fiscale, and that the one generated automatically through the Bocconi application and the Universitaly application is invalid.
