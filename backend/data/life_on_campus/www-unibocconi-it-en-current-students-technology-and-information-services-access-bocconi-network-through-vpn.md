# Access to the Bocconi network through VPN

(Service only available for Teachers and Technical-administrative staff)

A **VPN**, **Virtual Private Network**, is a private network activated between individuals, using the Internet as a public means of communication. Thanks to a VPN access server, a communication channel is created through the Internet, which carries encrypted and thus secure information.

The user can therefore use the internal network and it's services from home or from any internet station as he were physically operating from the Campus.

In addition, users that access Bocconi email through client applications (eg: Microsoft Outlook) can set the SMTP Bocconi server (smtp.unibocconi.it) as the default outgoing mail server when authenticating in the **VPN**.

To use this service, users need to have the necessary credentials (username and password).

Authentication is made by the LDAP protocol.

**The service is available only for Teachers and Technical-administrative staff**. The new VPN client software and the installation instructions are available by the [Professor's Diary](https://agenda.unibocconi.it/ "PROFESSOR'S AGENDA").

For any problems or for further information, please send an email to the following address: [noc@unibocconi.it](mailto:noc@unibocconi.it)
