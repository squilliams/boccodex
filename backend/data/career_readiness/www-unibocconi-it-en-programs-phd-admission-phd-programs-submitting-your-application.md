Starting your Application

Applications must be submitted exclusively through the on-line portal which is available [**here**](http://www.unibocconi.eu/admissionsportal) from 17th April 2026 18:00 CET. The deadline for submitting your application is 18th May 2026 at 23:59 CET. This deadline is not negotiable.

The application is free.

For technical issues, please contact us at [phdadmission@unibocconi.it](mailto:phdadmission@unibocconi.it)

In order to complete your application you will have to:

- register to obtain a Personal ID and password (Bocconi students already have the required credentials);
- enter the requested information in the spaces provided;
- upload the required documents;
- submit the application.

**Entering data and uploading documents:**

You can access the online procedure multiple times.

_**Entering data**_ **:**

- Enter data in each section and click SAVE.
- Data can be modified at any time and saved again by clicking SAVE

_**Uploading documents:**_

- You will upload documents to each separate section; If needed, an uploaded document can be replaced (the new one will be saved automatically) or deleted if you no longer wish to upload it.
- If more than one document is required, create a ZIP file and upload it.

**Submitting application:**

**First step:**

You should click on SUBMIT DATA only when all sections are complete and no further changes are needed. After submitting, data can no longer be modified

**Second step:**

Click on SUBMIT APPLICATION only when no further changes are needed in the UPLOADS sections. After submitting, neither data nor uploaded documents can be modified.

**In summary, you must click on SUBMIT DATA and SUBMIT APPLICATION only when you have finalized your application.**

**Applications are completed and can only be assessed after clicking on SUBMIT DATA and SUBMIT APPLICATION.**

After submitting the application, the icon APPLICATION RECEIPT will appear and you will be able to print out the application receipt, that you should keep.

Should you need information about data and/or documents you need to submit, contact [phdadmission@unibocconi.it](mailto:phdadmission@unibocconi.it)
