# Iscritti nelle Università milanesi anni accademici dal 2017 al 2025

Il dataset espone, relativamente all'anno accademico 2017-25 gli iscritti agli atenei milanesi suddivisi per classi di corso di studio e per genere.

La licenza con cui il dataset è rilasciato è la Italian Open Data License v2.0 ( [http://www.dati.gov.it/content/italian-open-data-license-v20](http://www.dati.gov.it/content/italian-open-data-license-v20))
